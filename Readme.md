# SupDeskNIT

Um sistema de gerenciamento de chamados construído com React no frontend e Node.js/Express no backend.

## Funcionalidades
- Abertura de chamados
- Acompanhamento e atualização de status
- Autenticação de usuários (Cliente e Técnico)

## Tecnologias
- **Frontend:** React, JavaScript, HTML, CSS
- **Backend:** Node.js, Express.js
- **Banco de Dados:** PostgreSQL
- **Controle de Versão:** Git, GitHub
- **Outros:** Deploy em nuvem, Testes, Acessibilidade, Uso de API

## Como Rodar (Em Construção)

## 🐳 Como Rodar com Docker

Este projeto utiliza **Docker** para padronizar o ambiente de desenvolvimento. Isso significa que não precisa instalar o PostgreSQL ou o Node.js localmente.

### Pré-requisitos
* [Docker Desktop](https://www.docker.com/products/docker-desktop/) instalado e rodando.

### Passo a Passo

1. **Clone o repositório e entre na pasta**
2. **Suba os containers:**

   No terminal, na raiz do projeto, execute:

     docker-compose up -d --build

3. **Acesse o sistema:**

   Frontend: http://localhost:5173

   Backend (API): http://localhost:5000

   Banco de Dados: Porta 5432 (PostgreSQL)
